# 珠江公园户外瑜伽



## 珠江公园户外瑜伽 2016年4月2日**

这次户外瑜伽参加的人数不多，但教练好几个，其他的都是资深高手。阿爽喉咙又上火了，讲话吃力，而荫生植物区里的小水塘里，好多青蛙叫得可欢了……总之“亚历山大”。

幸亏昨天录了一段引导词，伙伴们又都比较熟悉体式。

虽是阴天，但心情非常晴朗，多谢梦梦、静子、安妮、美苑、蓉蓉、百合的支持和鼓励，希望大家厚积薄发，更上一层楼。

**Namaste!**
